# Ollama's Documentation

Ollama is the easiest way to get up and running with large language models such as gpt-oss, Gemma 3, DeepSeek-R1, Qwen3 and more.

- **Quickstart**: Get up and running with your first model or integrate Ollama with your favorite tools
- **Cloud**: Ollama's cloud models offer larger models with better performance.
- **API reference**: View Ollama's API reference

## Libraries
- **Ollama's Python Library**: The official library for using Ollama with Python
- **Ollama's JavaScript Library**: The official library for using Ollama with JavaScript or TypeScript.
