# Accessibility Features

Our platform is designed for everyone. Accessibility options are always available in the header:

## Options
- **Large Font**: Increases font size for easier reading.
- **High Contrast**: Boosts color contrast for low-vision users.
- **Dyslexia Friendly**: Uses a dyslexia-friendly font and letter spacing.

## How to Use
- Click the accessibility icon in the header.
- Select your preferred mode. Changes apply instantly.
- Keyboard and screen reader friendly.

## Persistence
- Your selection remains active during your session.

## Feedback
- We welcome suggestions to improve accessibility. Contact support from the footer.
