# Search & Navigation

## Global Search
- Press `Ctrl+K` (Windows/Linux) or `Cmd+K` (Mac) to open the global search dialog from anywhere.
- Start typing to filter pages instantly.
- Use arrow keys to navigate results, `Enter` to select.
- Intuitive responses: friendly welcome, suggestions if no results.
- Keyboard and screen reader accessible.

## Navigation Structure
- Main navigation is always available in the header and footer.
- All major pages (Home, Features, Docs, API, Support, etc.) are one click away.
- Footer includes documentation and screenshot links.

---

# Accessibility

## Accessibility Toggle
- Always visible in the header for all users.
- Options:
  - **Large Font**: Increases font size for readability.
  - **High Contrast**: Improves color contrast for low-vision users.
  - **Dyslexia Friendly**: Uses a dyslexia-friendly font and letter spacing.
- Changes apply instantly and persist during your session.
- Works with keyboard navigation and screen readers.

---

# Onboarding

- New users are guided by intuitive navigation and search.
- Accessibility options are highlighted on first visit.
- Documentation and support are always accessible from the footer.
