# Buildings.com

Buildings.com is a modern platform for property management, communication, and services. Manage, connect, and scale your building operations efficiently.

## Features
- Commercial and residential solutions
- Tenant hub for requests, amenity booking, and notices
- Knowledge graph with schemas and API
- Security protocols and compliance
- Flexible pricing plans
- Documentation and developer resources

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/BuildingAi-Cloud/super-octo-rotary-phone.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
